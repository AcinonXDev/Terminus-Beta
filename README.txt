# Terminus #
A command-line file and system manager, still in development. Visual Studio 2017 or above reccomended to edit. Application stored in ./Terminus_<dev/release>_<version>/com.acinonDigital.Terminus/bin/<debug/release, depends on version>
You may alternatively use the NullSoft installer
****FEATURES****
-Fast access to files anywhere on your drive.
-Run applications from file paths
-Usable as an alternative to Explorer. *WARNING* This application does not have the same safety features as Windows Explorer.
-A customizable terminal, with basic commands for editing it.
-Simple system administration features
